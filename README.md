# 🎬 S1Player — Универсальный видеоплеер с поддержкой URL

**S1Player** — это мощный и простой в использовании HTML5-видеоплеер, созданный на основе [MediaElement.js](https://www.mediaelementjs.com/). Он поддерживает потоковое вещание HLS, MP4 и другие форматы, а также предоставляет удобные элементы управления и возможность загрузки контента по прямой ссылке из адресной строки.

![HTML](https://img.shields.io/badge/HTML-29.4%25-orange)

## ✨ Ключевые возможности

-   **Универсальность:** Поддерживает MP4, WebM, HLS (m3u8) и другие форматы.
-   **Гибкая загрузка контента:** Плеер по умолчанию показывает демо-поток «Россия 1», но вы можете легко заменить его на любой другой, передав параметр `src` в URL.
-   **Полноэкранный режим:** Поддерживается как встроенными средствами браузера, так и через API плеера.
-   **Кастомизация:** Включает в себя плагины качества (`quality`) и другие модули MediaElement.js.
-   **Водяной знак (Watermark):** Отображается при наведении курсора на плеер.
-   **Аналитика:** Интегрирован счётчик Яндекс.Метрики для отслеживания просмотров.

    Загрузка контента через параметр src
Главная особенность плеера — возможность загружать любой видео- или аудиоконтент по прямой ссылке.

Чтобы воспроизвести видео по своему URL, добавьте к адресу страницы параметр ?src= и укажите прямую ссылку на медиафайл или HLS-поток.

Примеры использования:

MP4 файл:
https://twixoffltdco.github.io/s1player/?src=https://example.com/video.mp4

HLS поток (m3u8):
https://twixoffltdco.github.io/s1player/?src=https://example.com/stream.m3u8

Плеер автоматически определит MIME-тип по расширению файла и запустит воспроизведение.

⚙️ Настройка и интеграция
Вы можете изменить внешний вид и поведение плеера, отредактировав файл index.html.

🔧 Основные параметры конфигурации
Настройки плеера передаются через объект new MediaElementPlayer('#my-video', { ... }):

autoplay: false — Автоматическое воспроизведение.

features — Массив, определяющий порядок и набор кнопок на панели управления (например, playpause, progress, volume, fullscreen, quality).

success: function(media) { ... } — Колбэк, который выполняется после успешной инициализации плеера. Внутри него можно обрабатывать события плеера, такие как play, pause, ended и другие.

💧 Водяной знак
По умолчанию в левом верхнем углу при наведении курсора на плеер отображается водяной знак. Вы можете заменить его, изменив ссылку на изображение в блоке:

html
<div class="watermark">
    <a href="https://russtube.ru" target="_blank">
        <img src="https://russtube.ru/1111.png" alt="watermark logo">
    </a>
</div>
